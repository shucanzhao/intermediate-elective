# General information

Author: Shucan Zhao.

Date: May 15, 2026

This repository contains submission for ENVS 193DD's Intermediate elective option 2. 
This work create a single advanced visualization, using one of a set of Tidy Tuesday submissions as graphic inspiration. 
I chose Cédric Scherer's Bill Dimensions of Brush-tailed Penguins and adapted his scatter.
